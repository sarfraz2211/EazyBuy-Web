for camera and gallery implementation with React native  in Ios add this into pblist 

<key>NSCameraUsageDescription</key>
<string>This app needs access to your camera to take photos for registration.</string>
<key>NSPhotoLibraryUsageDescription</key>
<string>This app needs access to your photo library to select images for registration.</string>
<key>NSPhotoLibraryAddUsageDescription</key>
<string>This app needs permission to save photos.</string>

User Guide for RN Works on Windows.
A) Windows setup for React Native + TypeScript (Android local)

Install

Git

Node.js LTS 20.x

JDK 17 (Temurin or Microsoft)

Android Studio with:

Android SDK Platform 34 or 35

Build-Tools 34.x

Platform-Tools

Command-line Tools (latest)

Android Emulator

2. Set environment variables (PowerShell as Admin; adjust paths):
   $env:JAVA_HOME="C:\Program Files\Eclipse Adoptium\jdk-17"  # or your JDK path
[Environment]::SetEnvironmentVariable("JAVA_HOME",$env:JAVA_HOME,"Machine")

$env:ANDROID_HOME="$env:LOCALAPPDATA\Android\Sdk"
[Environment]::SetEnvironmentVariable("ANDROID_HOME",$env:ANDROID_HOME,"Machine")

$path=[Environment]::GetEnvironmentVariable("Path","Machine")
$add=";$env:ANDROID_HOME\platform-tools;$env:ANDROID_HOME\emulator;$env:ANDROID_HOME\cmdline-tools\latest\bin"
[Environment]::SetEnvironmentVariable("Path",$path+$add,"Machine")

3. Close and reopen the terminal, then verify:

node -v
java -version
adb --version

4. Create a TypeScript RN app

npx @react-native-community/cli@latest init MyApp --template react-native-template-typescript
cd MyApp


5. Run Android

6. Create an AVD in Android Studio (e.g., Pixel 7 API 34).

emulator -list-avds
emulator -avd Pixel_7_API_34
npx react-native run-android

Rent a Mac (MacStadium)

Use this if you want full Xcode control and iOS Simulator.

Steps

Rent a Mac mini or VM.

On the Mac (via VNC/RDP):

Install Xcode from App Store. Open once to finish components.

Install Homebrew, Node 20, Watchman, CocoaPods:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install node@20 watchman cocoapods


Clone your repo. In project root:

npm ci
cd ios && pod install && cd ..


Open ios/MyApp.xcworkspace in Xcode, set Team, fix signing, choose a simulator, Run.

Archive and export .ipa for TestFlight or Ad Hoc.
